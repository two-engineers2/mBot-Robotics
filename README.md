# mBot-Robotics
A 5 day summer camp with step by step instructions and explanations to program mBot. 

Modules include: 
  - Motors
  - Infrared remote
  - Line follower sensor
  - Buzzer
  - Ultrasonic sensor
  - Combined ultrasonic sensor and line follower challenges
  - Sumo bot challenge.
